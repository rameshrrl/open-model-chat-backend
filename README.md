# AI App Backend

This is a basic TypeScript Node.js Express project with a single `/chat` route.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run in development mode:
   ```sh
   npm run dev
   ```
3. Build and start:
   ```sh
   npm run build
   npm start
   ```
